    This is the main DLL source file.

AdaptiveHuffman.h
    This file contains a class declaration for an Adaptive Huffman Encoder/Decoder.

CharList.cpp
	This is a source file for a helper class.

CharList.h
	This file contains a class declaration for a stripped down Linked List. This list is 
	used to hold data for the creation of new characters that are not yet pressent in the 
	Huffman struture.

AssemblyInfo.cpp
	Contains custom attributes for modifying assembly metadata.
